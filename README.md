# 📸 Photographer Portfolio Website

A modern, responsive photographer portfolio built using Next.js. This project showcases galleries, videos, and includes an admin panel to manage content dynamically.

---

## 🚀 Features

- 📷 Beautiful image gallery display
- 🎥 Video showcase section
- 🔐 Admin panel for content management
- ⚡ Fast and optimized with Next.js
- 📱 Fully responsive design
- 🎨 Clean UI with reusable components

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React.js
- TypeScript
- Tailwind CSS
- PostCSS

---

## 📁 Project Structure

photographer-portfolio/
│
├── app/                # App router pages (Next.js)
├── components/         # Reusable UI components
├── hooks/              # Custom React hooks
├── lib/                # Utility functions / helpers
├── public/             # Static assets (images/videos)
├── styles/             # Global styles
│
├── middleware.ts       # Middleware (auth/routing logic)
├── next.config.mjs     # Next.js config
├── tsconfig.json       # TypeScript config
├── postcss.config.mjs  # PostCSS config
├── components.json     # UI config (if using shadcn/ui)
├── package.json        # Dependencies
├── package-lock.json   # Lock file
├── .gitignore

## ⚙️ Installation

### 1. Clone the repository

git clone https://github.com/abhikr11/photographer-portfolio.git  
cd photographer-portfolio

### 2. Install dependencies

npm install

---

## ▶️ Run the Project

### Development mode

npm run dev

App will run on:
http://localhost:3000
